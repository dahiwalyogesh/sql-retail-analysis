# Retail Sales Analysis

## Overview
SQL analysis of retail sales data across products, categories and regions.

## Questions Answered
1. Total revenue per category
2. Total quantity and revenue per product
3. Categories with revenue above £20
4. Sales performance labelling (High/Medium/Low)
5. Top performing region by total revenue

## SQL Concepts Used
GROUP BY, HAVING, CASE WHEN, CTEs, Aggregate Functions

## Tool
SQLite — DB Browser
